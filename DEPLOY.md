# Deploy to GitHub Pages (Free)

Your website URL will be: **https://workwithG.github.io**

## Steps (10 minutes)

### 1. Create a GitHub account (if you don't have one)
Go to github.com → Sign up → use username **ashokgorantla** (or similar)

### 2. Create the repository
- Go to github.com/new
- Repository name: **ashokgorantla.github.io**  
  (must be exactly: YOUR-USERNAME.github.io)
- Set to **Public**
- Click "Create repository"

### 3. Upload the file
- On the new repo page, click "uploading an existing file"
- Drag and drop `index.html` from this folder
- Commit message: "Add personal website"
- Click "Commit changes"

### 4. Enable GitHub Pages
- Go to repository Settings → Pages (left sidebar)
- Source: Deploy from a branch
- Branch: **main** / **(root)**
- Click Save

### 5. Wait ~2 minutes
Your site will be live at **https://ashokgorantla.github.io**
